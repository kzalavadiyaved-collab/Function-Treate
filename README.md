# 📖 Overview

**Functional Treat** is an interactive Python application that performs different data analysis and transformation operations on one-dimensional and two-dimensional lists.

The project demonstrates important Python concepts including recursion, lambda expressions, built-in functions, user-defined functions, sorting, global variables, and many more.

# ⚡ Python Concepts Covered

| Feature | Description |
|----------|-------------|
| Built-in Functions | len(), sum(), min(), max(), sorted() |
| User Defined Functions | Custom functions for every operation |
| *args | Accept unlimited positional arguments |
| **kwargs | Display dataset summary |
| __doc__ | Function documentation |
| Lambda | Anonymous functions |
| Recursion | Factorial & Fibonacci |
| Global Keyword | Shared dataset information |
| Multiple Return Values | Return min, max, average together |
| Sorting | Ascending & Descending |
